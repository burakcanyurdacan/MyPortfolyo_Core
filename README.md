Dinamik Portfolyo & CV Yönetim Sistemi
<hr/>
Bu proje, Murat Yücedağ hocamın Udemy'deki eğitim serisini temel alarak hazırladığım dinamik CV yönetim ve portfolyo projesidir.
Proje sadece CV arayüzü olarak kalmayıp, verilerin dinamik olarak Admin panelinden kod müdahalesi olmadan rahat bir şekilde yönetilebildiği bir proje halini almıştır.
<br/><br/>
Proje Hakkında

Dinamik Portfolyo Projesi 2 bölümden oluşmaktadır:

1- Kullanıcı Arayüzü (UI)
Kişinin yeteneklerini, projelerini, referanslarını görüntüleyebildiği ve mesaj bırakabildiği bir yapıdır.

2- Admin Paneli
Bu panel üzerinden;
- Proje Listeleme / Ekleme / Silme / Güncelleme
- Yetenekleri yönetme
- Mesajları okuma ve durumlarını güncelleme
- Referansları Listeleme / Ekleme / Silme / Güncelleme

Gibi işlemler yapılabilmektedir.
<br/><br/>
Kullanılan Teknolojiler:
- HTML, CSS, BOOTSTRAP
- Microsoft SQL Server, LinQ
- ASP.NET Core 6.0
- Entity Framework Code First
